# Toast Notification🔔

A simple and responsive **Toast Notification** web application built using **HTML, CSS, and JavaScript**. This project displays modern toast messages for different notification types such as success, error and warning, providing users with quick and visually appealing feedback. Toast notifications appear temporarily on the screen and automatically disappear after a short duration. Toast notifications are commonly used in modern web applications to provide non-intrusive feedback to users.

## ⚙️ How It Works

1. User clicks a notification button.
2. JavaScript dynamically creates a toast message.
3. The toast appears with an animation.
4. After a few seconds, the toast automatically disappears.
5. Multiple notifications can be displayed without interrupting the user experience. ([CSS Script][1])
